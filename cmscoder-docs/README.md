# aicoder 文档

本目录用于承接 cmscoder 的总纲、专题设计、开发计划与进度文档，形成可持续演进的标准 spec 结构。

## 目录

- `specs/`：系统总纲、专题设计、开发计划与进度文档

## 使用方式

1. [specs/00-cmscoder-overview.md](./specs/00-cmscoder-overview.md) 作为项目总纲，不直接替代专题设计文档。
2. 新需求先更新或新增 `specs/` 下对应专题，再进入研发拆分。
3. 跨专题变更先更新 [specs/00-cmscoder-overview.md](./specs/00-cmscoder-overview.md)，再同步相关专题。
